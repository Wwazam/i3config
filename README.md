# i3config
myi3config

### Dependencies

```
scrot
xclip
telegram
```
**for theming:**

```
lxappearance
gtk-chtheme
qt4-qtconfig
```

### Extra setup

* Make Telegram executable from the Terminal: `sudo ln -s /opt/telegram/telegram /usr/local/bin/telegram`

* Change theme in `lxappearance` `gtk-chtheme` and `qt4-qtconfig`
